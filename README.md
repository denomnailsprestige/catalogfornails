# Denom Nails Prestige Catalog Prototype

This is a simple static catalog website for Instagram bio traffic. It is ready for GitHub Pages and does not need a domain, backend, cart, or payment system.

## Edit Before Publishing

- In `script.js`, replace `WHATSAPP_NUMBER = ""` with your WhatsApp number in international format, for example `923001234567`.
- In `index.html`, replace the placeholder product names, prices, size details, and descriptions.
- Replace images in `assets/` with final product photos when they are ready.
- Keep the main page named `index.html` so GitHub Pages opens it automatically.

## Publish On GitHub Pages

1. Create a public GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, `README.md`, and the `assets` folder.
3. Open repository settings.
4. Go to Pages.
5. Choose deploy from branch, then select the main branch and root folder.
6. Use the GitHub Pages website link in your Instagram bio.

The customer-facing link will look like:

```text
https://yourusername.github.io/repository-name/
```
